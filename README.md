# Task 2: Authentication Testing

## Objective
To understand and analyze authentication mechanisms in a web application.

## Website Tested
OWASP Juice Shop (Demo Website)

## Tools Used
- OWASP ZAP
- Web Browser

## Testing Performed
1. Checked login with invalid credentials
2. Observed error messages
3. Tested multiple failed login attempts

## Findings
- Clear error messages shown for wrong credentials
- No account lockout after multiple failures
- Weak password policy suspected
